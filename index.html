01 Euler Tour là gì?

Euler Tour (hay còn gọi là Euler Tour of a Tree / DFS Order) là kỹ thuật "trải phẳng" một cây có gốc thành một mảng 1 chiều bằng cách thực hiện DFS và ghi lại thứ tự vào/ra của mỗi đỉnh.

💡 Ý tưởng cốt lõi
Mỗi đỉnh u có hai chỉ số: tin[u] (thời điểm vào) và tout[u] (thời điểm ra). Toàn bộ cây con gốc u nằm gọn trong đoạn [tin[u], tout[u]] trên mảng.

Nhờ tính chất này, các truy vấn trên cây con có thể chuyển thành truy vấn đoạn thông thường – dễ dàng xử lý với Segment Tree, BIT, …

03 Thuật toán – DFS Order

  1. Khởi tạo bộ đếm thời gian timer = 0
  2. Khi DFS vào đỉnh u: gán tin[u] = ++timer, lưu order[timer] = u
  3. Đệ quy vào tất cả con của u
  4. Khi thoát khỏi u: gán tout[u] = timer
euler_tour.cpp 
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5 + 5;
vector<int> adj[MAXN];  // danh sách kề
int tin[MAXN], tout[MAXN], order[MAXN];
int timer_ = 0;

void dfs(int u, int parent) {
    tin[u] = ++timer_;
    order[timer_] = u;   // mảng thứ tự DFS

    for (int v : adj[u]) {
        if (v != parent)
            dfs(v, u);
    }

    tout[u] = timer_;  // kết thúc cây con u
}

// Kiểm tra u có phải tổ tiên của v không
bool isAncestor(int u, int v) {
    return tin[u] <= tin[v] && tout[v] <= tout[u];
}

int main() {
    int n; cin >> n;
    for (int i = 1; i < n; i++) {
        int u, v; cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    dfs(1, 0);  // gốc = 1, parent giả = 0
    return 0;
}
Thời gian O(N)
Bộ nhớ O(N)

04 Ứng dụng: Truy vấn cây con

Mỗi đỉnh u có giá trị val[u]. Hai thao tác:

⚡ Bài toán
Update: thêm x vào tất cả đỉnh trong cây con gốc u.
Query: tổng giá trị cây con gốc u.

Với Euler Tour, hai thao tác này trở thành Range Update / Range Query trên BIT:

subtree_query.cpp 
// BIT hỗ trợ range update, range query
long long bit[MAXN];
int n;

void update(int i, long long v) {
    for (; i <= n; i += i & -i) bit[i] += v;
}

long long query(int i) {
    long long s = 0;
    for (; i > 0; i -= i & -i) s += bit[i];
    return s;
}

// Cộng x vào toàn bộ cây con gốc u
void addSubtree(int u, long long x) {
    update(tin[u], x);
    update(tout[u] + 1, -x);
}

// Lấy tổng cây con gốc u
long long sumSubtree(int u) {
    return query(tout[u]) - query(tin[u] - 1);
}

05 Ứng dụng: LCA với Euler Tour

Có một biến thể Euler Tour khác dùng để tìm LCA (Lowest Common Ancestor): ghi lại mỗi lần DFS ghé thăm một đỉnh (kể cả lần quay lại), tạo ra mảng dài 2N−1 phần tử.

  1. Khi vào đỉnh u, đẩy u vào mảng tour.
  2. Sau khi thăm mỗi con v, đẩy lại u vào mảng.
  3. LCA(u, v) = đỉnh có độ sâu nhỏ nhất trong đoạn [first[u], first[v]].
  4. Dùng Sparse Table để trả lời RMQ trong O(1).
euler_lca.cpp 
vector<int> euler;          // mảng Euler Tour (2N-1)
int first[MAXN], depth[MAXN];

void dfs_lca(int u, int p, int d) {
    first[u] = (int)euler.size();
    depth[u] = d;
    euler.push_back(u);
    for (int v : adj[u]) {
        if (v == p) continue;
        dfs_lca(v, u, d + 1);
        euler.push_back(u);  // quay lại u sau mỗi con
    }
}

// RMQ bằng Sparse Table → LCA(u,v) trong O(1)
int lca(int u, int v) {
    int l = min(first[u], first[v]);
    int r = max(first[u], first[v]);
    return rmq(l, r); // đỉnh depth nhỏ nhất trong [l,r]
}
Preprocessing O(N log N)
Query LCA O(1)

07 Ứng dụng thực tiễn

🌲

Subtree Query

Tổng / min / max cây con với BIT hoặc Segment Tree.

🔗

LCA

Tổ tiên chung gần nhất trong O(1) qua RMQ + Sparse Table.

📏

Path Query

Truy vấn trên đường đi u→v kết hợp HLD hoặc Euler Tour 2 lần.

🔄

Rerooting

Đổi gốc cây động – dễ xử lý khi tin/tout đã có sẵn.

📦

DSU on Tree

Small-to-large / dsu on tree chạy O(N log N) nhờ Euler Order.

⚡

HLD

Heavy-Light Decomposition phụ thuộc Euler Tour để map cạnh nặng.

08 Lưu ý khi cài đặt

⚠ Cẩn thận
Với cây lớn (N ≈ 10⁵), DFS đệ quy có thể gây stack overflow. Hãy tăng stack size hoặc dùng DFS iterative (ngăn xếp tường minh).
iterative_dfs.cpp 
void euler_iterative(int root) {
    stack<pair<int,int>> st;   // {node, parent}
    st.push({root, -1});
    while (!st.empty()) {
        auto [u, p] = st.top(); st.pop();
        if (u > 0) {            // lần vào
            tin[u] = ++timer_;
            st.push({-u, p});  // đánh dấu lần ra
            for (int v : adj[u])
                if (v != p) st.push({v, u});
        } else {               // lần ra
            tout[-u] = timer_;
        }
    }
}
